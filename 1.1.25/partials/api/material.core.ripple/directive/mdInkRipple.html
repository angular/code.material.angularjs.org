The md-ink-ripple directive allows you to specify the ripple color or if a ripple is allowed.

Usage

String values

Ripples in red Not rippling

Interpolated values

Ripples with the return value of 'randomColor' function Ripples if 'canRipple' function return value is not 'false' or '0'

Attributes

Parameter Type Description
* md-ink-ripple stringboolean stringboolean

A color string #FF0000 or boolean (false or 0) for preventing ripple
