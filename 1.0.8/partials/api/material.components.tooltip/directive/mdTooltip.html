Tooltips are used to describe elements that are interactive and primarily graphical (not textual).

Place a <md-tooltip> as a child of the element it describes.

A tooltip will activate when the user focuses, hovers over, or touches the parent.

Usage

Play Music

Attributes

Parameter Type Description
md-visible expression expression

Boolean bound to whether the tooltip is currently visible.
md-delay number number

How many milliseconds to wait to show the tooltip after the user focuses, hovers, or touches the parent. Defaults to 300ms.
md-autohide boolean boolean

If present or provided with a boolean value, the tooltip will hide on mouse leave, regardless of focus
md-direction string string

Which direction would you like the tooltip to go? Supports left, right, top, and bottom. Defaults to bottom.
