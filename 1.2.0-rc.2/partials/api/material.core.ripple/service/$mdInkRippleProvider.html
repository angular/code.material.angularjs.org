If you want to disable ink ripples globally, for all components, you can call the disableInkRipple method in your app's config.

Usage

app.config(function ($mdInkRippleProvider) { $mdInkRippleProvider.disableInkRipple(); });

Methods


  • $mdInkRippleProvider.disableInkRipple();

    A config-time method that, when called, disables ripples globally.
