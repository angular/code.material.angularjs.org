Indicates that the associated element should be used as the delete button template for all chips. The associated element must be a child of md-chips.

The provided button template will be appended to each chip and will remove the associated chip on click.

The button is not styled or themed based on the theme set on the md-chips component. A theme class and custom icon can be specified in your template.

You can also specify the type of the button in your template.
