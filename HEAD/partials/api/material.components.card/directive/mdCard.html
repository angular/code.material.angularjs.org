The <md-card> directive is a container element used within <md-content> containers.

An image included as a direct descendant will fill the card's width, while the <md-card-content> container will wrap text content and provide padding. An <md-card-footer> element can be optionally included to put content flush against the bottom edge of the card.

Action buttons can be included in an <md-card-actions> element, similar to <md-dialog-actions>. You can then position buttons using layout attributes.

Cards have constant width and variable heights; where the maximum height is limited to what can fit within a single view on a platform, but it can temporarily expand as needed.
