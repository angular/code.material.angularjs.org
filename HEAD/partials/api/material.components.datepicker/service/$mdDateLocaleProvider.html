The $mdDateLocaleProvider is the provider that creates the $mdDateLocale service. This provider that allows the user to specify messages, formatters, and parsers for date internationalization. The $mdDateLocale service itself is consumed by Angular Material components that deal with dates.

Usage

myAppModule.config(function($mdDateLocaleProvider) { // Example of a French localization. $mdDateLocaleProvider.months = ['janvier', 'février', 'mars', ...]; $mdDateLocaleProvider.shortMonths = ['janv', 'févr', 'mars', ...]; $mdDateLocaleProvider.days = ['dimanche', 'lundi', 'mardi', ...]; $mdDateLocaleProvider.shortDays = ['Di', 'Lu', 'Ma', ...]; // Can change week display to start on Monday. $mdDateLocaleProvider.firstDayOfWeek = 1; // Optional. $mdDateLocaleProvider.dates = [1, 2, 3, 4, 5, 6, ...]; // Example uses moment.js to parse and format dates. $mdDateLocaleProvider.parseDate = function(dateString) { var m = moment(dateString, 'L', true); return m.isValid() ? m.toDate() : new Date(NaN); }; $mdDateLocaleProvider.formatDate = function(date) { var m = moment(date); return m.isValid() ? m.format('L') : ''; }; $mdDateLocaleProvider.monthHeaderFormatter = function(date) { return myShortMonths[date.getMonth()] + ' ' + date.getFullYear(); }; // In addition to date display, date components also need localized messages // for aria-labels for screen-reader users. $mdDateLocaleProvider.weekNumberFormatter = function(weekNumber) { return 'Semaine ' + weekNumber; }; $mdDateLocaleProvider.msgCalendar = 'Calendrier'; $mdDateLocaleProvider.msgOpenCalendar = 'Ouvrir le calendrier'; // You can also set when your calendar begins and ends. $mdDateLocaleProvider.firstRenderableDate = new Date(1776, 6, 4); $mdDateLocaleProvider.lastRenderableDate = new Date(2012, 11, 21); });

Properties


  • $mdDateLocaleProvider.months

    Array of month names (in order).

  • $mdDateLocaleProvider.shortMonths

    Array of abbreviated month names.

  • $mdDateLocaleProvider.days

    Array of the days of the week (in order).

  • $mdDateLocaleProvider.shortDays

    Array of abbreviated dayes of the week.

  • $mdDateLocaleProvider.dates

    Array of dates of the month. Only necessary for locales using a numeral system other than [1, 2, 3...].

  • $mdDateLocaleProvider.firstDayOfWeek

    The first day of the week. Sunday = 0, Monday = 1, etc.

  • $mdDateLocaleProvider.parseDate

    Function to parse a date object from a string.

  • $mdDateLocaleProvider.formatDate

    Function to format a date object to a string. The datepicker directive also provides the time zone, if it was specified.

  • $mdDateLocaleProvider.monthHeaderFormatter

    Function that returns the label for a month given a date.

  • $mdDateLocaleProvider.monthFormatter

    Function that returns the full name of a month for a giben date.

  • $mdDateLocaleProvider.weekNumberFormatter

    Function that returns a label for a week given the week number.

  • $mdDateLocaleProvider.msgCalendar

    Translation of the label "Calendar" for the current locale.

  • $mdDateLocaleProvider.msgOpenCalendar

    Translation of the button label "Open calendar" for the current locale.

  • $mdDateLocaleProvider.firstRenderableDate

    The date from which the datepicker calendar will begin rendering. Note that this will be ignored if a minimum date is set. Defaults to January 1st 1880.

  • $mdDateLocaleProvider.lastRenderableDate

    The last date that will be rendered by the datepicker calendar. Note that this will be ignored if a maximum date is set. Defaults to January 1st 2130.
