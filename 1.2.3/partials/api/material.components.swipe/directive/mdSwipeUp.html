The md-swipe-up directive allows you to specify custom behavior when an element is swiped up.

Notes

  • The $event.currentTarget of the swiped element will be null, but you can get a reference to the element that actually holds the md-swipe-up directive by using $target.current

You can see this in action on the demo page (Look at the Developer Tools console while swiping).
