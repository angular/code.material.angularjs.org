Service to announce messages to supported screenreaders.

The $mdLiveAnnouncer service is internally used for components to provide proper accessibility.

module.controller('AppCtrl', function($mdLiveAnnouncer) { // Basic announcement (Polite Mode) $mdLiveAnnouncer.announce('Hey Google'); // Custom announcement (Assertive Mode) $mdLiveAnnouncer.announce('Hey Google', 'assertive'); });

Methods


  • $mdLiveAnnouncer.announce(message, politeness);

    Announces messages to supported screenreaders.

    Parameter Type Description
    * message string string

    Message to be announced to the screenreader
    * politeness 'off''polite''assertive' 'off''polite''assertive'

    The politeness of the announcer element.
