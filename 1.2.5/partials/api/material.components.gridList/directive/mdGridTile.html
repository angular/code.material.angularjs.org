Tiles contain the content of an md-grid-list. They span one or more grid cells vertically or horizontally, and use md-grid-tile-{footer,header} to display secondary content.

Responsive Attributes

The md-grid-tile directive supports "responsive" attributes, which allow different md-rowspan and md-colspan values depending on the currently matching media query.

In order to set a responsive attribute, first define the fallback value with the standard attribute name, then add additional attributes with the following convention: {base-attribute-name}-{media-query-name}="{value}" (ie. md-colspan-sm="4")
