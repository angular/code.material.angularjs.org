{{currentDoc.name}} API Documentation

A service used to control inserting and removing of an element from the DOM. It is used by $mdBottomSheet, $mdDialog, $mdToast, $mdMenu, $mdPanel, and $mdSelect.

Methods


  • $$interimElementProvider.$service.show(options);

    Adds the $interimElement to the DOM and returns a special promise that will be resolved or rejected with hide or cancel, respectively.

    Parameter Type Description
    * options object object

    map of options and values
    Returns Description
    Promise

    a Promise that will be resolved when hide() is called or rejected when cancel() is called.

  • $$interimElementProvider.$service.hide(reason, options);

    Removes the $interimElement from the DOM and resolves the Promise returned from show().

    Parameter Type Description
    * reason * *

    Data used to resolve the Promise
    * options object object

    map of options and values
    Returns Description
    Promise

    a Promise that will be resolved after the element has been removed from the DOM.

  • $$interimElementProvider.$service.cancel(reason, options);

    Removes the $interimElement from the DOM and rejects the Promise returned from show().

    Parameter Type Description
    * reason * *

    Data used to resolve the Promise
    * options object object

    map of options and values
    Returns Description
    Promise

    Promise that will be resolved after the element has been removed from the DOM.
