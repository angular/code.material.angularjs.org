Displays a label separating groups of md-option element directives in a md-select box's dropdown menu.

Note: When using md-select-header element directives within a md-select, the labels that would normally be added to the md-optgroup directives are omitted, allowing the md-select-header to represent the option group label (and possibly more).

With label elements

Attributes

Parameter Type Description
