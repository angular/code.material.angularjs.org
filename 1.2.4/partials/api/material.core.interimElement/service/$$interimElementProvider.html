Factory that constructs $$interimElement.$service services. Used internally in material design for elements that appear on screen temporarily. The service provides a promise-like API for interacting with the temporary elements.

app.service('$mdToast', function($$interimElement) { var $mdToast = $$interimElement(toastDefaultOptions); return $mdToast; });
